TAG HTML
ลำดับที่ TAG รูปแบบการใช้งาน ตัวอย่างการใช้งาน
1 < !DOCTYPE html > กำหนดว่าเอกสารนี้เป็นเอกสาร HTML5 < !DOCTYPE html >
2 < html >,< /html > จุดเริ่มต้นและจุดสิ้นสุด ของการเขียนโค้ด < html >........< /html >
3 < head >,< /head > เป็นคำสั่งเบื้องหลังของเว็บไซต์ < head >........< /head >
4 < body >,< /body > กำหนดเนื้อหาของเอกสารและเป็นที่เก็บเนื้อหาที่มองเห็นได้บ้านหน้าเว็บ < body >........< /body >
5 < title >,< /title > กำหนดหัวเรื่องของเอกสารที่แสดงบนเบราว์เซอร์ < title >........< /title >
6 < p >,< /p > กำหนดย่อหน้า ขึ้นบรรทัดใหม่ < p >........< /p >
7 < hr > กำหนดการเปลี่ยนแปลงเฉพาะเรื่องในเนื้อหา < hr size = 2px noshade="">
8 < br > ขึ้นบรรทัดใหม่ < br >
9 < pre >,< /pre > กำหนดข้อความที่จัดรูปแบบไว้ล่วงหน้า < pre >.........< /pre >
10 < b >,< /b > and < strong >,< /strong > กำหนดตัวอักษรหนา < b >....< /b > , < strong >....< /strong >
11 < i >,< /i > and < em >,< /em > กำหนดตัวอักษรตัวเอียง < i >....< /i > , < em >....< /em >
12 < small >,< /smal > กำหนดข้อความที่เล็กกว่า < small >........< /smal >
13 < mark >,< /mark > กำหนดข้อความที่ทำเครื่องหมาย / เน้น < mark >........< /mark >
14 < u >…< /u > (< ins >..< /ins >) กำหนดข้อความบางส่วนที่ไม่มีการเคลื่อนไหวและมีสไตล์แตกต่างจากข้อความปกติ < u >........< /u >
15 < sub >,< sub > & < sup>,< sup > กำหนดข้อความที่ห้อย < sub >.....< sub > & < sup>....< sup >
16 < ul >,< /ul > & < li >, กำหนดรายการที่ไม่เรียงลำดับ < ul >.....< /ul > & < li >....< / li >
17 < ol >,< /ol > กำหนดรายการ < ol >........< /ol >
18 < img src = ""> เพิ่มรูปภาพ < img src = ‘xxx.png’ >
19 < !--...-- > กำหนดความคิดเห็น < !--...-- >
20 < a >,< /a > กำหนดไฮเปอร์ลิงก์ < a >www. .....< /a >
21 < table >,< /table > กำหนดตาราง < table >.........< /table >
22 < tr >,< /tr > กำหนดเซลล์ส่วนหัวในตาราง < tr >........< /tr >
23 < td >,< /td > กำหนดเซลล์ในตาราง < td >........< /td >
24 < th >,< /th > กำหนดเซลล์ส่วนหัวในตาราง < th >........< /th >
25 < caption >,< /caption > กำหนดคำอธิบายตาราง < caption >........< /caption >
26 < style > < /style > กำหนดข้อมูลสไตล์สำหรับเอกสาร < style >........< /style >
27 Background Color กำหนดสีพื้นหลังสำหรับองค์ประกอบ HTML Background Color:
28 "font-family: " กำหนดฟอนต์ตัวอักษร "font-family: "
29 "font-size: " กำหนดขนาดตัวอักษร "font-size: "
30 "text-align: center;" กำหนดตำแหน่งของตัวอักษร "text-align: center;"
31 < div >,< /div > กำหนดส่วนในเอกสาร < div >........< /div >
32 < span >,< /span > กำหนดส่วนในเอกสาร < span >........< /span >
33 < form >,< /form > กำหนดรูปแบบ HTML สำหรับการป้อนข้อมูลของผู้ใช้ < form >........< /form >
34 < label > กำหนดป้ายกำกับสำหรับองค์ประกอบ input < label >
